# Pomysł na projekt: FocusFlow

## Krótki opis
FocusFlow to prosta aplikacja webowa do organizacji dnia metodą Pomodoro. Użytkownik może tworzyć listę zadań, uruchamiać timer pracy i przerw oraz śledzić, ile sesji udało się ukończyć.

## Główne funkcje (MVP)
- Dodawanie, edycja i oznaczanie zadań jako ukończone.
- Timer Pomodoro (np. 25 minut pracy + 5 minut przerwy).
- Licznik ukończonych sesji dla każdego zadania.
- Zapis danych lokalnie w przeglądarce (Local Storage).

## Technologie
- HTML, CSS, JavaScript (Vanilla JS)
- Opcjonalnie: prosty framework CSS (np. Tailwind lub Bootstrap)

## Dlaczego ten projekt?
Projekt jest niewielki, ale praktyczny i pozwala przećwiczyć kluczowe umiejętności front-endowe:
- pracę z DOM,
- zarządzanie stanem aplikacji,
- obsługę zdarzeń i czasu,
- przechowywanie danych po stronie klienta.
