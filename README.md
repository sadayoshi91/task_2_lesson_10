# FocusFlow

FocusFlow to mała aplikacja webowa wspierająca pracę metodą Pomodoro.
Pozwala planować zadania, mierzyć sesje skupienia i śledzić postęp dnia.

## Funkcje MVP
- Lista zadań: dodawanie, edycja i oznaczanie jako ukończone.
- Timer Pomodoro: domyślnie 25 minut pracy i 5 minut przerwy.
- Licznik sesji: zliczanie ukończonych pomodoro dla zadań.
- Lokalny zapis danych: Local Storage w przeglądarce.

## Technologie
- HTML
- CSS
- JavaScript (Vanilla JS)

## Plan rozwoju

- Ciemny motyw interfejsu.
